How to Use Biasbuster

A step-by-step guide to detecting and addressing bias in your content

Getting Started

Biasbuster helps you identify and address potential bias in text content. Here's how to get started:

  • Click the Biasbuster icon in your browser toolbar
  • Select text on any webpage
  • Use keyboard shortcuts for quick access

Analyzing Text

Try selecting this text to see how Biasbuster works!

To analyze text for bias:

  1. Select any text on a webpage
  2. Right-click and choose "Analyze for Bias"
  3. Or use the shortcut: Alt + B
