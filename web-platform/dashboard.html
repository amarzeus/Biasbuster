Quick Start

Here's a simple example of how to use the Biasbuster API:

fetch('https://api.biasbuster.com/v1/analyze', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer bb_api_3f7a9c2e6b5d4801a2e5f8g7h6j5k4l3m2'
  },
  body: JSON.stringify({
    text: "Your article or text content here",
    options: {
      includeSentiment: true,
      includeCredibility: true
    }
  })
})
.then(response => response.json())
.then(data => console.log(data));

How to Create Effective Bias Patterns

Be Specific

Target specific words and phrases that consistently indicate bias rather than broad concepts.

Consider Context

Remember that context matters - some terms may only show bias in specific contexts.

Test Your Patterns

After creating patterns, test them on various texts to ensure they're detecting what you intended.

Avoid Overgeneralization

Be careful not to create patterns that flag neutral language or common terms as biased.
