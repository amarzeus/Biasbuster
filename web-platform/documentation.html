BiasBuster Documentation

Welcome to the BiasBuster documentation. This guide will help you understand and utilize our AI-powered bias detection platform effectively.

Quick Start Guide

1. Install the Extension

Download and install the BiasBuster Chrome extension from the Chrome Web Store.

2. Create an Account

Sign up for a free account to access all features and save your analysis history.

3. Start Analyzing

Click the BiasBuster icon while reading any news article to analyze it for bias.

Bias Detection

How It Works

BiasBuster uses advanced natural language processing and machine learning to identify various types of bias in text:

  • Political bias
  • Gender bias
  • Racial bias
  • Cultural bias
  • Socioeconomic bias

Analysis Process

  1. Text extraction and preprocessing
  2. Sentiment analysis
  3. Bias pattern recognition
  4. Context evaluation
  5. Source credibility assessment

API Reference

Authentication


POST /api/auth/register
POST /api/auth/login
GET /api/auth/me

Analysis


POST /api/analysis/analyze
GET /api/analysis/{id}
GET /api/analysis/user/analyses

Example Request


curl -X POST https://api.biasbuster.com/analysis/analyze \
-H "Authorization: Bearer YOUR_TOKEN" \
-H "Content-Type: application/json" \
-d '{"text": "Your text here"}'

Best Practices

For Users

  • Always verify results with multiple sources
  • Consider context and intent
  • Report false positives to improve accuracy
  • Use the educational resources provided

For Developers

  • Implement rate limiting for API calls
  • Cache analysis results when appropriate
  • Handle errors gracefully
  • Follow security best practices
