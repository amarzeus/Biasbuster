Installation

npm install biasbuster-extension

Follow these steps to get started with Biasbuster:

  1. Install the Chrome extension from the Chrome Web Store
  2. Create an account on our platform
  3. Configure your API keys and preferences

Quick Start Guide

1. Basic Setup

Install and configure the extension in your Chrome browser.

// Example configuration
{
  "apiKey": "your-api-key",
  "sensitivity": "medium",
  "features": ["realtime", "suggestions"]
}

API Reference

Analyze Text

POST /api/analyze

Request Body

{
  "text": "string",
  "options": {
    "sensitivity": "high" | "medium" | "low",
    "features": ["bias", "sentiment", "suggestions"]
  }
}
