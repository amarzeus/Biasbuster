Getting Started

What is Biasbuster?

Biasbuster is an AI-powered tool that detects, explains, and helps mitigate bias in news articles and media content. Our system uses advanced natural language processing and machine learning algorithms to analyze text for various types of bias.

Quick Start Guide

  1. Create an account on the Sign Up page
  2. Navigate to your dashboard after logging in
  3. Paste any article or text you want to analyze
  4. Click "Analyze" to receive a comprehensive bias analysis

API Reference

Base URL

https://api.biasbuster.com/v1

Authentication

All API requests require an API key which you can generate from your account dashboard.

Authorization: Bearer YOUR_API_KEY

Endpoints

POST /analyze

Analyzes text for bias and returns detailed results.

Request Body: 
{
  "text": "Your article or text content here",
  "options": {
    "includeSentiment": true,
    "includeCredibility": true
  }
}
Response: 
{
  "MainTopic": "Politics",
  "BiasAnalysis": {
    "OverallBias": "Left-leaning",
    "BiasScore": 2.4,
    "Details": [...]
  },
  "SentimentAnalysis": {...},
  "CredibilityScore": {...}
}
GET /perspectives

Retrieves alternative perspectives on a topic.

Query Parameters:
  • topic - The main topic to find perspectives on
  • count - Number of perspectives to return (default: 3)

Visualizations & Custom Bias Detection

Visualizations

Biasbuster provides rich visualizations to help understand bias in content:

  • Bias Distribution - Shows the distribution of different bias types detected in the text
  • Sentiment Over Time - Displays how sentiment changes throughout the text
  • Top Biased Phrases - Highlights the most significant biased phrases detected

Custom Bias Patterns

Biasbuster allows you to define custom bias patterns to detect language that you consider biased:

  1. Navigate to the Dashboard and select "Custom Bias Detection"
  2. Create patterns using regular expressions to match specific words or phrases
  3. Assign bias types and severity levels to your patterns
  4. Apply these patterns to your analysis for personalized bias detection

API Support for Custom Patterns

{
  "text": "Your article text here",
  "options": {
    "includeSentiment": true,
    "customBiasPatterns": [
      {
        "id": "my_custom_pattern",
        "pattern": "\\b(word1|phrase2|term3)\\b",
        "biasType": "my_bias_category",
        "severity": "2",
        "explanation": "This pattern detects specific terminology I consider biased"
      }
    ]
  }
}

Browser Extension

Installation

The Biasbuster browser extension is available for:

  • Chrome Web Store
  • Firefox Add-ons
  • Edge Add-ons

Usage

Once installed, the extension adds a Biasbuster icon to your browser toolbar. When viewing a news article or any page with substantial text content:

  1. Click the Biasbuster icon in your toolbar
  2. The extension will analyze the main content of the page
  3. View the bias analysis directly in the popup

Bias Categories

Biasbuster detects and categorizes bias into several main categories:

Political Bias

Bias that favors a particular political ideology, party, or figure. This includes left-leaning, right-leaning, or other political orientations.

Framing Bias

How information is presented to influence interpretation through selective presentation of facts or context.

Source Bias

Reliance on sources that have their own biases or agendas, or selective citation of sources.

Emotional Bias

Use of emotional language or appeals to influence the reader's perception.

Omission Bias

Leaving out important information that could provide a more complete perspective.

Implementation Guide

Integrating Biasbuster

For developers looking to integrate Biasbuster into their applications:

JavaScript Example
// Using fetch API
const analyzeBias = async (text) => {
  const response = await fetch('https://api.biasbuster.com/v1/analyze', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': 'Bearer YOUR_API_KEY'
    },
    body: JSON.stringify({
      text: text,
      options: {
        includeSentiment: true,
        includeCredibility: true
      }
    })
  });
  
  return await response.json();
};
Node.js Example
const axios = require('axios');

async function analyzeBias(text) {
  try {
    const response = await axios.post('https://api.biasbuster.com/v1/analyze', {
      text: text,
      options: {
        includeSentiment: true,
        includeCredibility: true
      }
    }, {
      headers: {
        'Content-Type': 'application/json',
        'Authorization': 'Bearer YOUR_API_KEY'
      }
    });
    
    return response.data;
  } catch (error) {
    console.error('Error analyzing bias:', error);
    throw error;
  }
}

Frequently Asked Questions

How accurate is Biasbuster's bias detection?

Biasbuster achieves approximately 85-90% accuracy in detecting bias compared to human evaluators. Our system is continuously trained on diverse sources and regularly updated to improve accuracy.

What languages are supported?

Currently, Biasbuster fully supports English content, with beta support for Spanish, French, and German. We're actively expanding our language capabilities.

Can Biasbuster analyze video or audio content?

At this time, Biasbuster primarily analyzes text content. For video or audio, you would need to provide a transcript for analysis.

How is user data handled?

We prioritize user privacy. Content submitted for analysis is not permanently stored unless explicitly requested by the user. See our Privacy Policy for more details.
